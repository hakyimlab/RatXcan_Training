Previously, we computed heritability (h2) estimates for rat genes using GEMMA. And our PredictDB pipeline computes predictive performance (R2) for each gene when training the EN model. Now we compare it to heritability and predictability estimates for human genes generated by Wheeler Lab.

The sqlite database genarch.db contains all results from Wheeler Lab’s multi-tissue analysis of GTEx data. We queried R2 and h2 values and stored them in human_h2.csv.
